# AI Simulation & LLM Evaluation Portfolio

## Overview
This repository showcases structured work in AI training, evaluation, and simulation-oriented system design.

## Included Work

### FSM Repair & Normalization
A deterministic system for validating, repairing, and normalizing finite state machines under strict constraints. Demonstrates structured reasoning, constraint enforcement, and reproducible output generation.

### Snorkel LLM Evaluation
Hands-on work involving programmatic labeling, structured task design, and LLM evaluation workflows.

### Synthetic Data Scenarios
Design of structured scenarios and edge cases for training and evaluating intelligent systems in controlled environments.

## Core Strengths Demonstrated
- Deterministic system design
- Structured data transformation
- AI training and evaluation workflows
- Edge-case handling and validation
- Simulation-oriented thinking
